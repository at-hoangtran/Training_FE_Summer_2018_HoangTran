What is `use strict`? what are the advantages and disadvantages to using it?

- Use strict: là từ khóa khai báo sử dụng chế độ "Strict Mode". Khi bạn muốn sử dụng Strict Mode ở đâu thì chỉ cần đặt "use strict" ở đó.
- Strict Mode giúp bạn viết mã Javascript dễ dàng an toàn hơn, thay đổi bad syntax đã được chấp nhận trước đó thành một lỗi thực sự.
- Lợi ích: - Giúp tránh được một số lỗi không đáng có trong Javascript, dễ dàng viết code Javascript an toàn, dễ đọc hơn.

What are the differences between `==` and `===`? Write an example for each case (if any)?

- "==": Chỉ so sánh giá trị.
Example:
+ 5 == 5 //true
+ 5 == "5" //true
- "===": So sánh cả giá trị và kiểu dữ liệu.
Example:
+ 5 === 5 //true
+ 5 === "5" //false

What is different between declaration: `var`, `const` and `let`?

*let và const chỉ sử dụng ở ES6.
- const: dùng để khai báo hằng số - là một giá trị không thay đổi trong suốt quá trình chạy.
- let: tạo ra biến chỉ có thể truy cập được trong block bao quanh nó.
- var: tạo ra biến có thể truy xuất trong function chứa nó. Ngoài ra, khi ở global scope, var tạo ra thuộc tính mới cho global object(this), còn let thì không.
